## Deskripsi Repository

Repository ini berisi kumpulan materi, konfigurasi, serta tugas praktikum Jaringan Komputer yang dikerjakan selama proses pembelajaran. Setiap modul mencakup implementasi dan dokumentasi sesuai topik praktikum yang telah diberikan.

- Modul 2 - Routing dan Implementasi IPv6
